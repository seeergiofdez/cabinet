With the new beggining of The BBJProjeK Organization network infrastructure, the online infra of The Lapiacery Project has been deprecated and will be migrated to the main bbjprojek website in the following months.

The Wi-Fi and internet network as well as of the intranet services of Lapiacery will be shut down in the following days too. The main BBJProjeK network has been improved with the Lapiacery network deprecation.

If you don't have keys for using the main BBJProjeK network, feel free to contact support@bbjprojek.org and provide your regular lapiacery network credentials so we can sign you up in the new and unified network infrastructure.

Happy weekend!
