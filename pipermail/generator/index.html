Pipermail Ticket Message Generator

Temporarily generate HTML for a pipermail ticket. (Until we automatize everything up)

  • On the content input, you can use "@@" to add line breaks.
  • Date should follow this layout: Mon Jan 8 16:00:00 UTC 2024.
  • Ticket and previous ticket name should not contain "[Ticket]" as it's already filled.
