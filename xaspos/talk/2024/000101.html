On January 2, I undertook the redaction of the new OpenStreetMap Madrid project; Tenería, on Pinto. Requesting cadastral data from the Tenería I, Tenería II and Parque Europa suburbs, for merging some of the data into the OpenStreetMap database using the Java OpenStreetMap Editor (JOSM) tool:

* A new project and import to use cadastral data, ensuring that it's properly formatted and aligned with OSM standards.

To enhance transparency and assist the OSM community, I documented the source of the cadastral data, including attribution and licensing information. The steps taken during the import process are also documented, creating a comprehensive record for future reference.

The progress can be watched on my cabinet post: https://cabinet.bbjprojek.org/blog/entries/2024/01/02/teneria_mapping

The intention for submitting this ticket is to provide a detailed account of the acvity related to this data import.
