As of March 1st, 2024, we're starting to support Windows 11 based machines.

In the following days and weeks, we are going to add new posts and resources for your new installations at our wiki web (https://wiki.bbjprojek.org). We will also release a full-length guide on installing and migrating every userdata to this new Windows version.

The Windows 11 versions supported:
* Windows 11 23H2 LTSC Preview (build 25941). Download link: https://archive.org/details/win11-ltsc-2024-25941

NOTE: To see every Windows supported versions on the bbjprojek core, check the following page:

        https://wiki.bbjprojek.org/docs/windows/config
