Important notice. We have renamed the primary branches for all repositories that used Periodic Semantic versioning (e.g., Thallium, Strontium, Redstone). This change standardizes our branch naming convention.

Some key changes:
The Thallium branch (the last stable version) has been renamed to master and is and will be the default branch.

Strontium, Redstone and other will be renamed to master-s, master-r, and master-x, respectively..

Actions Required
Before your next git pull or git push, you must update your local workspace to avoid errors. Please choose one of the following options. Recloning or updating local branch.

Cloning:
* git clone 

Updating branch:
* git branch -m Thallium master
* git fetch origin
* git branch --set-upstream-to=origin/master master

For questions or assistance, please contact the OPWG or leave your queries at #gitlab on Slack.
