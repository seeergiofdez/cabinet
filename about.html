About

The Authority (formerly Sergio's Personal Cabinet) is a website which hosts some documentation and projects made by a small group of the organization. Formerly, it was Sergio's personal site, which has moved to https://sergio.bbjprojek.org/. Here you will find some documentation about SilentFootPrint, and other iterations, such as The Authority or The Agency. This domain is also currently hosting XasPos↗.

Website

This website is built from scratch, all the HTML and CSS is manually written, and the whole repository is publicly available at bbjprojek/cabinet↗.

This website is intended to use really lightweight resources (with pages around 12-52 kB without images). There are several stylesheets and scripts so this way only the necessary features are provided for each HTML page. The main stylesheets information can be found on the README↗ file of the repository.

Cookies and tracking

This page does not store cookies. If you find one, it’s probably a bug, and we’d appreciate if you’d let us know.

To show some iframes and external info, a button is used to prevent non-consensual tracking as far as possible, these iframes are “hidden” behind the button. Note that we use non-tracking embed codes, yet they may still track you; this is beyond our control. This also saves a lot of bandwith, since the third-party iframes pull in a lot of JavaScript.

No analytics are recorded. This page is hosted on GitLab Pages. Also, this page is designed to last. Almost every principle that Jeff Huang specifies in their post is followed: This Page is Designed to Last↗. Vanilla HTML/CSS is used, as few pages as possible are written to retain information, native fonts, and trying no hotlink.
