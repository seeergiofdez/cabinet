Requests

If you're interest in an album poster not done before, I accept requests. Just send me the album name as well as other useful information via email at sergio@bbjprojek.org↗.

Before requesting an album poster, please take a look at the albums in my Pinterest gallery to check if the one you want is already made.

Poster Versioning

Since the first poster, there has been several versions and models:

  • Model I, II, III: Main one. Almost 200 posters using this model. It's based on other Pinterest posters. It includes Spotify QR, and has a carton color as background, Mostin typography as the heading and regular text.
  • Model IV: Has the same design as the previous ones, but it uses Sagona font as the regular text.
  • Model V: New modern design; featuring Lusitana↗ and La Belle Aurore↗ fonts for heading text, keeping our golden pick, Sagona, for regular text.
  • Model Smoothness: The project was for alternative music, so I made an alternative poster design. Featuring DM Sans↗ and Sagona↗ fonts; they were previously part of our BBJProjeK Coporate Typographies↗. Text will be always stylished lowercase.
