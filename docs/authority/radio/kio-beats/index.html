Kio Beats

"Let the game begin. Which one of these songs will you hate the most?"

Trivia

  • It's the station that was initially directed by The Ragyxo Foundation, but rights were moved to BBJProjeK, throught The Bureau iteration.
  • Kio Beats is based on Mineral Heights II.
  • An initial motto of this station can still be found at the public available ticket of the first revision of the radio: "Let the game begin. Which one of these songs will you hate the most?".
  • This station uses "Self Radio" GTA V commentaries, and features Jonathan Moore, as already seen on The Labs X FM (DLC 12). It features less adverts and commercials than previous radios.
