Trivia

  • This station uses "Self Radio" GTA V commentaries, featuring adverts and commercials. The remastered version also contains some BBJProjeK unique commentaries for song outros.
  • The radio internal files, a.k.a. codename, list this station as "ertms," a reference to the European Rail Traffic Management System (where this station has born; in a train). A security system known for its failures and stops, installed in all Civia trains.
  • This is one of the stations that doesn't feature as much commercials o as other ones.
  • The Labs X FM is your ticket to musical journeys that defy genres, taking you on a ride through uncharted sonic landscapes.
  • The Labs X FM doesn't just play music; it curate moods. Each track is a carefully selected potion, creating a vibe that's both eclectic and cohesive.
  • This is the "newest" radio station, being the oldest song released in 2013, which is an anachronism.
