The Labs X FM

"We don't fit in anywhere. We're better than anyone else. Listen to The Labs X FM"

The Labs X FM is a station composed of alternative, dream-pop and R&B ambiented music. Containing songs from the 2020s. The radio is being hosted by Cliff Lane and Andee.

Information

  • Location: Mineral Heights, Rabbau
  • Genre: Alternative, dream-pop and R&B
  • Songs: 36 (+ 170 on DLCs) See full song list
  • Frequency: 94.5 / 94.6 FM
  • Host: The BBJProjeK Organization

Trivia

  • This station uses "Self Radio" GTA V commentaries, featuring adverts and commercials. The remastered version also contains some BBJProjeK unique commentaries for song outros.
  • The radio internal files, a.k.a. codename, list this station as "ertms," a reference to the European Rail Traffic Management System (where this station has born; in a train). A security system known for its failures and stops, installed in all Civia trains.
  • This is one of the stations that doesn't feature as much commercials o as other ones. Also it's the "newest" radio station, being the oldest song released around the 2010s, which is an anachronism.
  • The radio production is financed by RaBBAu, a part of The Ragyxo Foundation.
