Releasing Radio Stations Optional Packages

A Radio Station Optional Package (DLC) consists of an optional MP3 file comprising 8-10, and occasionally up to 12 songs. It serves as a content enhancement released for existing main stations, maintaining the original format including commentaries and advertisements. The DLC aligns with the album metadata of the main radio stations, seamlessly integrating into the playlist rotation. Presently, each radio can accommodate up to 6 DLCs.

Right now, radio stations DLCs are only available as MP3, which means that there is no “online streaming” version. Therefore you won’t find any DLCs songs in any Spotify playlist.

June 2024 Update

From June 2024, DLCs can have now a maximum of 16 songs, keeping the other options available. This mean that as of now, optional packages can have 8, 10, 12 or 16 songs for each package.

DLCs are still only available as MP3.
