Radio Stations Guidelines

Radio Stations Internal Guidelines

Some internal guidelines before procedding to make a new radio station.

Files needed for a radio station

The following files are mandatory:

  • .aup3 = Audacity project
  • .mp3 = variable 260kbps mp3 or fixed 320kbps mp3
  • .png = Image of the logo, using logo guidelines at 1800px
  • .png = Image of the logo used for embedding in mp3 file, same image as previous one but at 800px
  • .zip = Containing the audio files (music only) used for the audacity project. Recommended to be FLAC files, but 320kbps MP3 are accepted.

Optionally, you can have "_adverts" folder which will contain any illustration used for advertisment purposes.

About Radio

Radio Types

There are currently 3 available station lengths:

  • Small: composed of 26 songs (~1h 30min) Radio Authority Park
  • Regular: composed of 36 songs (~2h 30min) The Labs X FM
  • Full: composed of 42 songs (~3h) Independence Pop FM

Exporting Modes

The audacity project file can be exported in several formats and quality:

  • On-The-Go: variable bitrate of about 220-260kbps in MP3
  • Regular: fixed bitrate of 320kbps in MP3
  • Lossless: FLAC file of 1441kbps (only use if downloaded songs are FLAC too)

NOTE: From January 1, 2024, any 128kbps or below bitrate radio won't be streamed.

Embedded artwork in audio file

Embedded artwork will always be a png file of 800x800px. This is to ensure the audio file is not heavier of the original audio.

Another mandatory metadata is the artist (The BBJProjeK Organization), and album and song name (“radio name”).
