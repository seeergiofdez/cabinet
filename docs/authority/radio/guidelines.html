Radio Stations Guidelines

Radio Station Internal Guidelines

Some internal guidelines before proceeding to make a new radio station, or make optional packages of a current one.

Files needed for a radio station

The following files are mandatory:

  • .aup3 = Audacity project
  • .mp3 = fixed 320kbps mp3 file
  • .png = Image of the logo, using logo guidelines at 1800px
  • .zip = Containing the audio files (music only) used for the audacity project. Recommended to be FLAC files, but 320kbps MP3 are also accepted.
  • .png = Image of the logo used for embedding in mp3 file, same image as previous one but at 800px

Optionally, you can have "_adverts" folder which will contain any illustration used for advertisment purposes.

About Radio

Radio Types

There are currently 3 available station lengths:

  • Small: composed of 26 songs (~1h 30min). Example: Radio Authority Park
  • Regular: composed of 36 songs (~2h 30min). Example: The Labs X FM
  • Full: composed of 42 songs (~3h). Example: Independence Pop FM

Exporting Modes

The audacity project file can be exported in several formats and quality:

  • Regular: fixed bitrate of 320kbps in MP3 (most used)
  • Lossless: FLAC file of 1441kbps (only use if downloaded songs are FLAC too)(optional)

Embedded artwork in audio file

Embedded artwork will always be a png file of 800x800px. This is to ensure the audio file is not heavier of the original audio.

Another mandatory metadata is the artist (The BBJProjeK Organization), and album and song name (“radio name”).

Mp3tag↗ program is used for setting up the embedded info into the mp3 file (artwork + track information).

Optional Packages

A Radio Station Optional Package (DLC) consists of an optional MP3 file comprising more songs of a current radio station. It serves as a content enhancement released for existing main stations, maintaining the original format including commentaries and advertisements.

The DLC aligns with the album metadata of the main radio stations, seamlessly integrating into the playlist rotation. Previously, due to infra limitation, each radio can accommodate up to 6 DLCs. We can host up to 120 DLCs for each radio now.

Radio stations DLCs will only be available as MP3, which means that there is no “online streaming” version. Therefore you won’t find any DLCs songs in any Spotify playlist.

Song numbers

From June 2024, DLCs can have now a maximum of 16 songs, keeping the other options available. This mean that as of now, optional packages can have 8, 10, 12, 14 or 16 songs for each package.
